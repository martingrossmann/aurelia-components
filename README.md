# Aurelia components
Common useful components for the Aurelia Framework (https://aurelia.io)

## Install

```shell script
yarn add t-systems-aurelia-components
```

## Usage

Import directly from sources
```typescript
import {CacheService} from 't-systems-aurelia-components'
```
